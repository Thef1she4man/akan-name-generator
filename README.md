# akan-name-generator
A simple web application that calculates the day of the week a user was born and assigns them an Akan name based on their gender. Akan names come from Ghanaian culture, where children are named based on the day they were born.

# Author
Aaron Michael Mitoko

## Setup Instructions
1. Clone the repo: `git clone https://github.com/Thef1she4man/akan-name-generator.git`
2. Open `index.html` in a browser.
